# Como usar o Conda

**Ative o ambiente virtual**

```bash
conda activate [ambiente]
```

**Instale os pacotes**

```bash
conda install pandas numpy scipy
```
ou
```bash
conda install -r requirements.txt
```

**Inicie o jupyter**

```bash
jupyter lab
```
